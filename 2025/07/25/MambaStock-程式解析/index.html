Mamba

主要的 Mamba 架構

  • 主要內容:
    • 每個 Mamba 經過 ResidualBlock 後，用 RMSNorm 正規化
  • forward(): 運用過去一段時間的資訊，預測未來一段時間的資訊
    • 輸入: 可以是過去 30 天的股價
    • 輸出: 可以是未來 5 天的股價
  • step(): 運用前一個時間點的資訊，預測下一個時間點的資訊
    • 輸入: 今天股價
    • 輸出: 明天股價

訓練參數

資料前處理

  • data = pd.read_csv(args.ts_code+'.SH.csv')
    輸入資料

  • data['trade_date'] = pd.to_datetime(data['trade_date'], format='%Y%m%d')
    日期

  • close = data.pop('close').values
    把 table 裡面要預測的目標 (close 每日收盤價的 column) 取出來 (原資料同時刪除該 column)，用變數 close 儲存

  • ratechg = data['pct_chg'].apply(lambda x:0.01*x).values
    資料裡面的投資報酬率 pct_chg 是百分比的資料 (e.g., 0.3%)，我們把投資報酬率的資料換回真實的值 (e.g., 0.3% -> 0.003)，用 ratechg 這個變數存下來。

  • data.drop(columns=['pre_close','change','pct_chg'],inplace=True)
    作者去除認為無用的 column

  • dat = data.iloc[:,2:].values
    前兩個 column 是 ts_code 和 trade_date 不用納入我們的資料。因此我們從第三個 (第二個如果從0開始數，也就是程式的寫法) 開始取資料，用變數 dat 儲存資料。

訓練資料 / 測試資料

  • trainX, testX = dat[:-args.n_test, :], dat[-args.n_test:, :]
    n_test 的值代表要保留多少當作測試資料。
    e.g., n_test=100 然後假如我們有 1000 筆資料 (1000天)，那 trainX 有前 900 筆資料，testX 有最後 100 筆資料。

  • trainy = ratechg[:-args.n_test]
    對應的於訓練資料的 y (投資報酬率)

  • predictions = PredictWithData(trainX, trainy, testX)
    將訓練資料 trainX、 trainy 以及測試資料 testX 作為 PredictWithData() 的輸入。

訓練結果呈現

  • time = data['trade_date'][-args.n_test:]
    測試資料的時間軸，存放在 time 這個變數

  • data1 = close[-args.n_test:]
    測試時間軸上的真實收盤價，存放在 data1 這個變數裡面

  • finalpredicted_stock_price = []
    放預測股價的空字串

  • pred = close[-args.n_test-1]
    一個 real number，是測試集最開始前一天的收盤價，也就是訓練集最後一天的收盤價。

  • for i in range(args.n_test):
    pred = close[-args.n_test-1+i]*(1+predictions[i])
    finalpredicted_stock_price.append(pred)
